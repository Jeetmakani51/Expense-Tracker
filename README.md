**Expense Tracker**

A simple web app to track your daily expenses, view totals, and see your top spending categories.

*Features*

-> Add expense with description, amount, category, and date

-> View expense history

-> See Total Expense, This Month's Spending, and Top Category

-> Delete expenses

-> Data is saved in localStorage, so it persists across page reloads

*How to Use*

-> Open index.html in your browser

-> Fill in the expense details in the form and click Add Expense

-> Your expenses will appear in the history section
